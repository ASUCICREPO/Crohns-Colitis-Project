This page demonstrates how the chat widget appears on your website.

Integration Instructions

To add this widget to your website, include this code before the closing </body> tag:

<script src="https://your-domain.com/widget.js" data-auto-init="true"></script>

Or initialize manually:

<script src="https://your-domain.com/widget.js"></script> <script> DisabilityRightsChatWidget.init(); </script>
